# Liftday — Claude Instructions

## Project
Next.js PWA fitness tracker. Stack: Next.js, TypeScript, Tailwind CSS v4, shadcn/ui.

## Workflow
- After completing any change, always commit and push (`git push origin main`) as the final step.
- Only push when 100% confident the changes are correct.

## Key Files
- `components/RestTimer.tsx` — circular rest timer with 3-2-1 countdown pulse
- `components/ExerciseScreen.tsx` — main workout screen
- `app/globals.css` — custom keyframes and global styles
- `lib/constants.ts` — shared constants (REST_DURATION, etc.)
